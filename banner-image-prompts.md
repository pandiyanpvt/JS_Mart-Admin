# JS Mart – Banner image generation prompts

ஒவ்வொரு பானருக்கும் AI image generator (DALL·E, Midjourney, Ideogram, etc.)-ல் use பண்ணலாம். Dimensions exact-ஆ வைத்துக்கொண்டு prompt copy-paste பண்ணி image generate பண்ணுங்க.

## Level 1: Header / Hero carousel (Homepage top)

**Dimensions:** 1920×600 px • **Aspect ratio:** 16:5

**Prompt (English):**
```
Wide horizontal e-commerce hero banner, 16:5 aspect ratio, modern grocery or retail store vibe. Clean, bright, minimal. Fresh produce or daily essentials arranged attractively. Soft lighting, no text or logos. Space in center for overlay text. Professional product photography style. High resolution, 1920x600.
```

**Prompt (Tamil context – grocery):**
```
Wide horizontal banner for online grocery store homepage. Fresh vegetables, fruits, and pantry items. Clean white and green tones. No text. Space for title overlay. 16:5 wide format, 1920x600.
```

## Tips

1. **No text in image:** Text overlay Admin/storefront-ல் தனியே போடுவதால், image-ல் text வராமல் “No text” / “Space for overlay” சொல்லுங்க.
2. **Resolution:** Prompt-ல் dimensions (e.g. 1920x600) சேர்த்தால் aspect ratio சரியா வரும்.
3. **Consistency:** “Clean, minimal, e-commerce, professional” மாதிரி words use பண்ணினால் எல்லா பானரும் ஒரே style-ல் இருக்கும்.

இந்த prompts-ஐ copy பண்ணி image generator-ல் paste பண்ணி generate பண்ணுங்க; need இருந்தால் [category/season] part மட்டும் மாற்றுங்க.
